Support

Use this page for Video Theater setup help, common troubleshooting, and contact options.

Common Checks

Make sure the page contains a video element, then open the Video Theater toolbar popup and move the sliders. Some browser pages, extension pages, and restricted pages cannot be adjusted by extensions.

If saved settings do not appear on another page, confirm the page uses the same hostname and that auto-apply is enabled in the options page.

Resetting Settings

Use Clear in the popup to remove one quick setting for the current domain. Use the options page to delete individual presets or delete every saved preset from the danger zone.

Report an Issue

Open a GitHub issue with your browser version, the affected website hostname, what you expected to happen, and what happened instead.
